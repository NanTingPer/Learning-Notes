## 1.1 函数的定义和定义域

函数: 与编程不一样的是，我们在面对数学函数时，关注的是他的输出。我们在编写方法时关注的是他的过程。而计算函数内容的过程，便是对数学函数的逆向过程。

### 一、函数三要素

1. x: 自变量 -> 定义域 -> x的取值范围，

   因为输入的内容为x，因此当x为某值时，导致此函数无意义，那么此值便不在x的取值范围内。

2. y: 因变量 -> 值域 -> y的取值范围

   因为输入内容而变化的值，当x的输入导致函数有意义时，y可能的范围。

3. f: 对应法则(一个程序上的方法定义) -> 映射关系 -> x和y的关系

   输入x输出y，参考.NET LINQ Select，我们需要计算的便是Select中给定的委托内容执行后，会输出什么。

   对于Select中的输入的`<TIn>`，便是x的定义域，将其限制在`<TIn>`的类型，因为其他类型都是无意义的，Select中的委托也无法处理这之外的内容。

   对于Select中的输出的`<TOut>`，便是y的值域，将其限制在`<TOut>`，因为你给我`<TIn>`我只能给你`<TOut>`。这也是 `In`和`Out`的关系。

### 1. 集合与区间表示法

- 条件 `x = 3`：  
  集合表示：{3} 或 {x | x = 3}  
  区间表示：[3, 3]

- 条件 `2 ≤ x ≤ 4`：  
  集合表示：{x | 2 ≤ x ≤ 4}  
  区间表示：[2, 4]

- 条件 `x > 0`：  
  集合表示：{x | x > 0}  
  区间表示：(0, +∞)

### 2. 常见函数的自然定义域

**自然对数函数**  
函数形式：ln(f(x))  
定义域条件：f(x) > 0  
定义域：解不等式 f(x) > 0

**平方根函数**  
函数形式：√(f(x))  
定义域条件：f(x) ≥ 0  
定义域：解不等式 f(x) ≥ 0

**反正弦 / 反余弦函数**  
函数形式：arcsin(f(x)) 或 arccos(f(x))  
定义域条件：-1 ≤ f(x) ≤ 1  
定义域：解不等式组 -1 ≤ f(x) ≤ 1

**分式函数**  
函数形式：1 / f(x)  
定义域条件：f(x) ≠ 0  
定义域：解方程 f(x) ≠ 0

**示例**：求 arcsin(x-3) 的定义域  
由 -1 ≤ x-3 ≤ 1，解得 2 ≤ x ≤ 4  
因此定义域为 [2, 4]

---

### 3. 从给定定义域反推函数（构造法）

已知目标定义域为闭区间 [a, b]，可以构造以下函数，使其自然定义域恰好等于 [a, b]：

- **平方根法**  
  $$
  y = \sqrt{(x-a)(b-x)}
  $$
  约束条件：(x-a)(b-x) ≥ 0 → a ≤ x ≤ b

- **反正弦法**  
  $$
  y = \arcsin\left(\frac{2x-(a+b)}{b-a}\right)
  $$
  约束条件：  
  $$
  -1 \le \frac{2x-(a+b)}{b-a} \le 1 \;\rightarrow\; a \le x \le b
  $$
  举例：a=2, b=4 时，函数为 y = arcsin(x-3)

- **根号和法**  
  $$
  y = \sqrt{x-a} + \sqrt{b-x}
  $$
  约束条件同平方根法，定义域为 [a, b]

### 4. 二次不等式的求解步骤

1. 将不等式化为标准形式 ax² + bx + c ≥ 0（或 ≤, >, <），使一边为 0。
2. 若 a < 0，两边同时乘以 -1，不等号方向反转。
3. 计算判别式 Δ = b² - 4ac：
   - 若 Δ < 0：二次式恒正（a>0）或恒负（a<0），直接判断解集为全体实数或空集。
   - 若 Δ = 0：解为 x = -b/(2a)（等号成立点）。
   - 若 Δ > 0：求出两根 x₁, x₂（x₁ < x₂），根据开口方向写出解集。

**示例**：解不等式 x - x² ≥ 2

步骤：  
$$
x - x^2 - 2 \ge 0
$$
$$
\Rightarrow -x^2 + x - 2 \ge 0
$$
$$
\Rightarrow x^2 - x + 2 \le 0 \quad (\text{乘以 }-1,\text{ 不等号反转})
$$

计算判别式：  
$$
\Delta = (-1)^2 - 4 \cdot 1 \cdot 2 = 1 - 8 = -7 < 0
$$
且 a = 1 > 0，所以 x² - x + 2 > 0 恒成立。  
因此 x² - x + 2 ≤ 0 无解。  
解集为空集：∅

---

### 5. 根号下表达式的化简与符号处理

关键等式：  
$$
6x - x^2 - 8 = -(x^2 - 6x + 8)
$$

不等号反转规则：若 A ≥ 0，则 -A ≤ 0。  
因此：  
$$
6x - x^2 - 8 \ge 0 \;\iff\; -(x^2 - 6x + 8) \ge 0 \;\iff\; x^2 - 6x + 8 \le 0
$$

注意：不能随意约去变量。例如，从 6x - x² ≥ 8 不能推出 x - x² ≥ 2，因为两边减去 5x 后右边会含有 x。

---

### 6. 一元二次不等式的因式分解法

对于形如 x² - 6x + 8 ≤ 0 的不等式：  
因式分解得  
$$
(x-2)(x-4) \le 0
$$
解集位于两根之间：2 ≤ x ≤ 4。

一般结论：若 p < q，则不等式  
$$
(x-p)(x-q) \le 0
$$
的解集为 p ≤ x ≤ q。

---

### 7. 平方根函数的值域与定义域

考虑函数  
$$
y = \sqrt{(x-2)(4-x)}
$$
- 定义域：由 (x-2)(4-x) ≥ 0 解得 2 ≤ x ≤ 4。  
- 值域：当 x = 3 时取最大值 1，最小值为 0，故值域为 [0, 1]。
