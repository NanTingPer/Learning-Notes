## 放置门解析

```cs
// i => 图格x坐标
// j => 图格y坐标
// type => 门类型
// style => 门样式
public static bool PlaceDoor(int i, int j, int type, int style = 0)
{
	int num = style / 36;
	int num2 = style % 36;
	int num3 = 54 * num;
	int num4 = 54 * num2;
	try {
        // 其实这里可以看到的是，他对比的传入 i,j 的上下两格的位置
        // 也就是 
        // j + 1 = 门，j + 2 = 实体块
        // j - 1 = 门，j - 2 = 实体块
        // 这符合门放置的条件
        
		if (Main.tile[i, j - 2].nactive() &&  // HasUnactuatedTile 判断是否存在未激活的图格
            Main.tileSolid[Main.tile[i, j - 2].type] &&  // 判断是否是实心块
            SolidTile(i, j + 2))  // 判断是否是实心块
        {
            // 设置图格活跃状态 这边可以看到，分别设置了 -1, j, +1的状态
            // 并且frameY是递增的
			Main.tile[i, j - 1].active(active: true); 
			Main.tile[i, j - 1].type = 10;
			Main.tile[i, j - 1].frameY = (short)num4;
			Main.tile[i, j - 1].frameX = (short)(num3 + genRand.Next(3) * 18);
			Main.tile[i, j].active(active: true);
			Main.tile[i, j].type = 10;
			Main.tile[i, j].frameY = (short)(num4 + 18);
			Main.tile[i, j].frameX = (short)(num3 + genRand.Next(3) * 18);
			Main.tile[i, j + 1].active(active: true);
			Main.tile[i, j + 1].type = 10;
			Main.tile[i, j + 1].frameY = (short)(num4 + 36);
			Main.tile[i, j + 1].frameX = (short)(num3 + genRand.Next(3) * 18);
			return true;
		}

		return false;
	}
	catch {
		return false;
	}
}
```



## 多图格放置解析

多图格数据无法通过`Tile`获取，而是经过`TileObjectData`，通过现有图格可以直接取得其数据

```cs
var mouseTile = Main.MouseWorld.ToTileCoordinates16();
var tile = Main.tile[mouseTile];
var tileObjectData = TileObjectData.GetTileData(tile);
```

---

如果要放置一个多图格，可以根据其具体样式选择`WorldGen`中的`Place`相关方法。若要自行放置，则可继续看。

在`TileObjectData`包含一个属性`Origin`，例 { 1, 1 }。以及`Height Width` 也就是高和宽。而`Origin`则表示，其放置时是以中心为坐标，向四周辐射。如表格所示。
$$
\begin{array}{c|c|c}
 &  &  \\
\hline
 & 轴心 &  \\
\hline
 & & 
\end{array}
$$

```cs
for (int x = width - 1; x >= 0; x--) {
    for (int y = height - 1; y >= 0; y--) {
        ...
    }
}
```



下面以门为例子，进行一个门的放置。

```cs
var mouseTile = Main.MouseWorld.ToTileCoordinates16();
var height = 3;
var width = 1;
// 以鼠标为中心，上2和下2需要具有实心块
if (WorldGen.SolidTile(mouseTile.X, mouseTile.Y + 2) &&
    WorldGen.SolidTile(mouseTile.X, mouseTile.Y - 2)) 
{
    for (int x = width - 1; x >= 0; x--) {
        for (int y = height - 1; y >= 0; y--) {
            var xCoord = mouseTile.X + x;
            var yCoord = mouseTile.Y + y;
            // 使用CanPlace可以为你判断是否能放置你需要放置的图格块
            if (TileObject.CanPlace(xCoord, yCoord, 10, 0, 1, out var obj)) {
                TileObject.Place(obj);
            }
        }
    }
}
```



## 制动器放置

如果在某个图格上拥有制动器，那么他的`HasActuator`就为true。其来自于`TileWallWireStateData`。

```cs
var mouseTile = Main.MouseWorld.ToTileCoordinates16();
var tile = Main.tile[mouseTile];
tile.HasActuator = true;

// 或者
var data = tile.Get<TileWallWireStateData>();
data.HasActuator = true;
```



## 箱子迁移

```cs
var mouseTile = Main.MouseWorld.ToTileCoordinates16();
// 获取鼠标位置下的箱子Id 拥有Id后可以去Main.chest获取箱子对象
var chestId = Chest.FindChest(mouseTile.X, mouseTile.Y);
var chest = Main.chest[chestId]; // 给定箱子
var chestItems = chest.item; // 全部物品

// 强制销毁箱子
Chest.DestroyChestDirect(mouseTile.X, mouseTile.Y, chestId);

// 放置新箱子，会返回新箱子的Id
var newChestId = WorldGen.PlaceChest(mouseTile.X, mouseTile.Y); ;
var newChestObj = Main.chest[newChestId];

// 将旧箱子的物品，覆盖到新箱子中
for (int i = 0; i < chestItems.Length; i++) {
    newChestObj.item[i] = chestItems[i];
}
```
